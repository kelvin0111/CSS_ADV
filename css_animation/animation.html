CSS Animations Examples

This example demonstrates a simple keyframe animation that changes the background color from blue to green.

This example shows an animation that rotates the element 360 degrees, with intermediate keyframes at 0, 50, and 100 percent.

This example demonstrates an animation that starts with a delay. The box fades in and moves to the right after a 1-second delay.

This example shows an animation that scales the element up and down infinitely.
