CSS Transitions Examples

Example 1: Hover Transition

This example demonstrates a transition that occurs when you hover over the button. The background color smoothly changes from blue to green.

Example 2: Focus Transition

This example shows a transition effect when the input field receives focus. The border color smoothly changes to blue.

Example 3: Active Transition

In this example, a transition is applied when the button is pressed. The button slightly scales down to give a pressing effect.

Example 4: Checked Transition

Here, a transition is applied when a checkbox is checked. The associated label changes color smoothly.

Example 5: JavaScript Triggered Transition

This example shows a transition triggered by JavaScript. When you click the button, the box rotates and changes color.

Explanation: Example 1: Hover Transition HTML: CSS: .example1 defines the initial style, and :hover defines the style change on hover. Transition: Changes background color smoothly when hovered over. Example 2: Focus Transition HTML: CSS: .example2 defines the initial style, and :focus defines the style change on focus. Transition: Changes border color smoothly when the input field is focused. Example 3: Active Transition HTML: CSS: .example3 defines the initial style, and :active defines the style change when pressed. Transition: Scales the button down smoothly when it is pressed. Example 4: Checked Transition HTML: CSS: .example4 + label defines the initial style, and :checked + label defines the style change when checked. Transition: Changes the label color smoothly when the checkbox is checked. Example 5: JavaScript Triggered Transition HTML:
CSS: .example5 defines the initial style, and .example5.active defines the style change. JavaScript: Toggles the active class on the box to trigger the transition. Transition: Rotates and changes the color of the box smoothly when the button is clicked.
